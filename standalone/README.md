## ğŸ¨ Customization

### Changing Colors

Edit the CSS variables in `css/style.css`:

```css
:root {
    --navy: #0a1628;           /* Primary dark color */
    --teal: #14b8a6;           /* Accent color */
    --cream: #faf8f5;          /* Background color */
    /* ... more variables */
}
```

### Updating Content

- **Personal Info**: Update name, title, and descriptions in each HTML file
- **Contact Details**: Replace email, phone, and social links
- **Projects**: Edit the project cards in `projects.html`
- **Services**: Modify service offerings in `services.html`

### Adding Images

- Replace image URLs with your own images
- Use local images by placing them in an `images/` folder
- Update the `src` attributes accordingly
